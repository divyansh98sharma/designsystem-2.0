# Stack

A flexbox layout primitive (`<ecw-stack>`). Arranges its projected children in a
`row` or `column` with a **token-driven `gap`** (spacing scale: 2, 4, 8, 12, 16)
and a **limited set of named `align`/`justify` roles**. It keeps spacing and
alignment on-system, so layouts stay consistent without hand-rolled CSS.

---

## Usage

```ts
import { EcwStackComponent } from '@ecw/design-system';

@Component({
  imports: [EcwStackComponent],
  template: `
    <ecw-stack direction="row" [gap]="8">
      <div>One</div>
      <div>Two</div>
      <div>Three</div>
    </ecw-stack>
  `,
})
export class MyComponent {}
```

Children are projected via content. Every visual property maps to a single
inline style on the host, so there is no template chrome beyond the projected
content.

---

## Guidelines

**Do**

- Use the `gap` scale (2, 4, 8, 12, 16) so spacing stays token-driven.
- Reach for the named `align`/`justify` roles (`start`, `center`, `end`,
  `stretch`/`between`) to express alignment intent.
- Pick `direction` by layout intent — `column` for vertical flow, `row` for
  horizontal.

**Don't**

- Don't hand-roll flexbox or margins to space children — that's exactly what the
  `gap` scale replaces.
- Don't reach for off-scale spacing values; the limited scale is intentional.
